# VkImageCreateFlags(3) Manual Page

## Name

VkImageCreateFlags - Bitmask of VkImageCreateFlagBits



## <a href="#_c_specification" class="anchor"></a>C Specification

``` c
// Provided by VK_VERSION_1_0
typedef VkFlags VkImageCreateFlags;
```

## <a href="#_description" class="anchor"></a>Description

`VkImageCreateFlags` is a bitmask type for setting a mask of zero or
more [VkImageCreateFlagBits](https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkImageCreateFlagBits.html).
