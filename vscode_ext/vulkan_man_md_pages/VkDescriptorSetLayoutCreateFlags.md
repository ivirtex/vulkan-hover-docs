# VkDescriptorSetLayoutCreateFlags(3) Manual Page

## Name

VkDescriptorSetLayoutCreateFlags - Bitmask of
VkDescriptorSetLayoutCreateFlagBits



## <a href="#_c_specification" class="anchor"></a>C Specification

``` c
// Provided by VK_VERSION_1_0
typedef VkFlags VkDescriptorSetLayoutCreateFlags;
```

## <a href="#_description" class="anchor"></a>Description

`VkDescriptorSetLayoutCreateFlags` is a bitmask type for setting a mask
of zero or more
[VkDescriptorSetLayoutCreateFlagBits](https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkDescriptorSetLayoutCreateFlagBits.html).
