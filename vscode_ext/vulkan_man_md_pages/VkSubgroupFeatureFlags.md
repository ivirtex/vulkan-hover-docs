# VkSubgroupFeatureFlags(3) Manual Page

## Name

VkSubgroupFeatureFlags - Bitmask of VkSubgroupFeatureFlagBits



## <a href="#_c_specification" class="anchor"></a>C Specification

``` c
// Provided by VK_VERSION_1_1
typedef VkFlags VkSubgroupFeatureFlags;
```

## <a href="#_description" class="anchor"></a>Description

`VkSubgroupFeatureFlags` is a bitmask type for setting a mask of zero or
more [VkSubgroupFeatureFlagBits](https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkSubgroupFeatureFlagBits.html).
