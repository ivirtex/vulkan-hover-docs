# VkVideoEncodeH264RateControlFlagsKHR(3) Manual Page

## Name

VkVideoEncodeH264RateControlFlagsKHR - Bitmask specifying H.264 encode
rate control flags



## <a href="#_c_specification" class="anchor"></a>C Specification

``` c
// Provided by VK_KHR_video_encode_h264
typedef VkFlags VkVideoEncodeH264RateControlFlagsKHR;
```

## <a href="#_description" class="anchor"></a>Description

`VkVideoEncodeH264RateControlFlagsKHR` is a bitmask type for setting a
mask of zero or more
[VkVideoEncodeH264RateControlFlagBitsKHR](https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkVideoEncodeH264RateControlFlagBitsKHR.html).
