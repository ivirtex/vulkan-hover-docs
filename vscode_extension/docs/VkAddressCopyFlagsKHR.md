# VkAddressCopyFlagsKHR(3) Manual Page

## Name

VkAddressCopyFlagsKHR - Bitmask of VkAddressCopyFlagBitsKHR



## [](#_c_specification)C Specification

```c++
// Provided by VK_KHR_copy_memory_indirect
typedef VkFlags VkAddressCopyFlagsKHR;
```

## [](#_description)Description

`VkAddressCopyFlagsKHR` is a bitmask type for setting a mask of zero or more [VkAddressCopyFlagBitsKHR](https://registry.khronos.org/vulkan/specs/latest/man/html/VkAddressCopyFlagBitsKHR.html).
