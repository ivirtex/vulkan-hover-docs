# VkPipelineColorBlendStateCreateFlags(3) Manual Page

## Name

VkPipelineColorBlendStateCreateFlags - Bitmask of VkPipelineColorBlendStateCreateFlagBits



## [](#_c_specification)C Specification

```c++
// Provided by VK_VERSION_1_0
typedef VkFlags VkPipelineColorBlendStateCreateFlags;
```

## [](#_description)Description

`VkPipelineColorBlendStateCreateFlags` is a bitmask type for setting a mask of zero or more [VkPipelineColorBlendStateCreateFlagBits](https://registry.khronos.org/vulkan/specs/latest/man/html/VkPipelineColorBlendStateCreateFlagBits.html).
