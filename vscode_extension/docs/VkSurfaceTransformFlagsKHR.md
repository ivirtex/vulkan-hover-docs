# VkSurfaceTransformFlagsKHR(3) Manual Page

## Name

VkSurfaceTransformFlagsKHR - Bitmask of VkSurfaceTransformFlagBitsKHR



## <a href="#_c_specification" class="anchor"></a>C Specification

``` c
// Provided by VK_KHR_display
typedef VkFlags VkSurfaceTransformFlagsKHR;
```

## <a href="#_description" class="anchor"></a>Description

`VkSurfaceTransformFlagsKHR` is a bitmask type for setting a mask of
zero or more
[VkSurfaceTransformFlagBitsKHR](https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkSurfaceTransformFlagBitsKHR.html).
